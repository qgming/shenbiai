# 更多内容

## 提示词管理

在提示词管理页面用户可以自由新增提示词，从而定制化个人的写作风格

软件内置了完整的提示词，可以直接查看，用户可以复制提示词再此基础上进行自由修改

![提示词](/images/software/prompt.png)

## 应用中心

应用中心页面内置多个功能按钮，点击即可进入相应的功能页面

在右下角标注浏览器表示的代表此功能会跳转到浏览器

![应用中心](/images/software/tool.png)

![书名生成器](/images/software/bookname.png)

## 用量统计

软件内有用量统计功能，可以统计总的调用次数，Token 使用量，各功能的使用量，每天的使用变化等信息

> 注意：软件内为预估 Token，以模型供应商的统计为准

![用量统计](/images/software/token.png)

## 数据设置

软件内包含数据备份恢复，清空所有数据功能

![数据](/images/software/data.png)

## 软件更新

在`关于我们`页面可以使用检查更新功能获取新版本

![更新](/images/software/about.png)
