# 神笔写作 - 专为网文创作设计的 AI 写作助手

## 产品简介

神笔写作是一款专为网络小说创作者设计的桌面端 AI 辅助写作软件。我们利用先进的大语言模型技术，为网文作者提供智能化的创作支持，从情节构思到章节创作，全方位提升您的写作效率。

**核心定位**：完全免费、数据本地存储、支持自定义 API 和提示词的网文创作工具。

## 核心特性

### 🖋️ 智能写作助手

- **章节细纲生成**：基于故事背景和人物设定，自动生成详细的章节大纲
- **正文内容创作**：根据细纲自动生成完整的章节正文
- **角色设定维护**：动态管理人物信息，确保角色一致性
- **情节发展建议**：提供合理的情节推进方案

### 📚 丰富提示词库

- **预设提示词模板**：提供章节细纲、正文创作、人物设定等多种创作场景
- **完全自定义支持**：用户可根据需求创建、编辑和管理专属提示词
- **场景化创作**：针对不同网文类型（玄幻、都市、言情等）优化提示词

![提示词](/images/software/prompt.png)

### 🌐 API 自定义接入

- **多模型支持**：兼容神笔 AI、OpenAI、DeepSeek 等多种 API 接口
- **灵活配置**：支持自定义 API 端点、密钥和参数设置
- **模型切换**：可根据创作需求灵活选择不同的大语言模型

### 🔨 丰富工具箱

- **AI 率检测**：智能分析文本 AI 生成比例
- **模型排行榜**：对比不同模型在写作场景下的表现
- **封面生成**：为作品自动生成精美封面
- **一键跳转**：快速访问常用工具，提升创作效率

![工具箱](/images/software/tool.png)

### 📱 优雅写作体验

- **深色/浅色模式**：支持界面主题切换，保护视力
- **自动保存**：实时保存创作内容，防止数据丢失
- **实时对话记录**：完整保存与 AI 的交互历史，便于回溯和参考

### 🔒 数据安全与隐私

- **完全本地存储**：所有书籍数据、创作内容均存储在本地设备
- **数据所有权**：创作内容完全属于用户，保障知识产权
- **离线使用**：核心功能支持离线操作，不受网络限制
