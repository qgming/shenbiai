# 基础配置

## 软件安装

下载神笔写作安装包，按照提示完成安装过程。

![安装](/images/software/install.png)

## 系统要求

- Windows 10/11 操作系统
- 4GB 以上内存
- 2GB 可用磁盘空间

## 首次启动

首次启动时需要进行模型供应商和功能模型的基本配置。

## 配置供应商

神笔写作软件支持多种模型供应商，您可以选择内置供应商或自定义供应商。

### 内置供应商

软件已内置以下供应商，您只需输入对应的 API Key 即可使用：

- **[神笔 AI](https://ai.qgming.com/)** - 官方推荐供应商
- **[DeepSeek](https://platform.deepseek.com/)** - 支持多种模型
- **[硅基流动](https://cloud.siliconflow.cn/i/BT2o4ohd)** - 高性能模型服务
- **[OpenAI](https://platform.openai.com)** - 兼容 GPT 系列模型

![供应商](/images/software/model.png)

### 自定义供应商

神笔写作软件完全兼容 OpenAI 接口，您可以配置任何支持 OpenAI 格式的模型服务。

**配置步骤：**

1. 在供应商列表中选择"自定义"
2. 输入供应商名称
3. 配置 API 端点（Base URL）
4. 输入 API Key
5. 测试连接并保存配置

![添加供应商](/images/software/addprovider.png)

## 配置功能模型

根据不同的写作功能，推荐配置相应的优化模型：

### 正文写作

- **推荐模型**：`gemini-2.5-pro`
- **备选模型**：`gpt-5`、`deepseek-v3.2`
- **说明**：需要高质量的文本生成能力，支持长文本输出，用于根据章节细纲和原文生成完整的新章节

### 基础模型

- **推荐模型**：`shenbi-base`
- **说明**：用于基础文本处理，如章节梗概等，推荐使用高性价比的中文模型

### 章节细纲

- **推荐模型**：`deepseek-v3.2` 或 `gemini-2.5-flash`
- **说明**：需要快速响应，用于通过对话根据自动提供的背景资料生成新章节章节细纲

### 章节评估

- **推荐模型**：`shenbi-base`
- **要求**：必须支持 JSON 格式输出
- **说明**：用于评估章节质量和结构合理性

### 设定维护

- **推荐模型**：`shenbi-base`
- **要求**：必须支持工具调用（Function Calling）
- **说明**：用于维护作品设定和世界观

### 嵌入模型

- **推荐模型**：`shenbi-embedding`
- **说明**：只支持配置嵌入模型，软件基于 1024 维度进行测试。如果使用其他维度模型，请确保进行兼容性测试。

### 生成器

- **推荐模型**：`gemini-2.5-pro`、`gpt-5`、`deepseek-v3.2`
- **说明**：推荐使用顶尖模型，确保高质量内容生成

## 配置验证

完成配置后，建议进行以下验证步骤：

1. **连接测试**：测试所有配置的模型供应商连接状态
2. **功能测试**：测试各功能模块的模型响应
3. **性能测试**：验证模型响应时间和输出质量

## 配置优化建议

- **性能优先**：对于需要快速响应的功能（如章节细纲），选择响应速度快的模型
- **质量优先**：对于正文写作等关键功能，选择输出质量高的模型
- **成本考虑**：根据使用频率和预算选择合适的模型组合

## 常见问题

**Q: 如何获取 API Key？**
A: 访问相应供应商的官方网站注册账号并获取 API Key。

**Q: 自定义供应商需要什么格式？**
A: 需要提供完整的 OpenAI 兼容接口

**Q: 嵌入模型为什么推荐 shenbi-embedding？**
A: 软件基于 1024 维度进行优化测试，使用其他维度模型可能需要进行额外测试。
